功能概要：K64_MQXLite工程
版权所有：苏州大学嵌入式中心(sumcu.suda.edu.cn)
版本更新：2016-04-01
硬件说明：PORTD3=红灯；PORTD4=蓝灯；PORTD5=绿灯

1. 工程概述
(1)本工程是针对苏大嵌入式实验室K64开发板设计的样例工程，本工程带有操作系统MQXLite。
(2)使用串口与外界通信，使用串口1，波特率为9600，1位停止位，无校验
(3)启动串口接收中断，回发接收数据

2. 软件环境
(1)Kinetis Design Studio IDE v3.0.0.

3. 硬件环境
(1)苏大嵌入式实验室K64开发板

4. 实验现象
(1)K64开发板红灯闪烁
(2)串口发送循环发送“light red”
